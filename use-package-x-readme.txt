This package provides the following extra keywords for
`use-package':

- :advice-add
- :advice-remove
- :custom-face*
- :custom-icon
- :doc
- :tag
- :hook-suffix
- :hook+
- :keymap-define
- :keymap-set
- :setopt
- :local-set
- :which-key-replacement
- :emacs<
- :emacs<=
- :emacs=
- :emacs>
- :emacs>=

All these keywords will autoload when used,
No need for "(require 'use-package-x-...)".
